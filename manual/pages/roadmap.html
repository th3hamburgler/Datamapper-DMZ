Road Map

DMZ is fully functional as is. Certain new features are in the queue to help make DMZ easier to use, as well as adding more extensions.

  • Add hooks for extensions, so they can directly modify core functions (like get, save, and delete).
  • Debugging Extension: An extension to aid in debugging DMZ models and queries.
  • Symmetric Relationships: Automatically create a relationship between B-and-A when A-and-B is saved.
  • Instant App Maker: Browser-based or standalone tool to generate all the code needed to create a simple application based on user input.
  • Some sort of intelligent inheritance system, whether through classes or some internal DMZ magic.

If you have any suggestions or ideas on what you'd like to see in DMZ, feel free to contact me!
